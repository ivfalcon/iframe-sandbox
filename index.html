Booking Iframe — Same Property

THN detects this page via the location pattern ^\\/\\w+\\/roomsandrates\\/iframe\\/[0-9]+\\/?$. The booking engine iframe below loads at /en/roomsandrates/1015553 which matches the "Rooms and Rates" page pattern for the child agent. Same property (1015553) = same siteId.

Booking Iframe — Other Property

Same top frame detection (Booking Iframe page), but the iframe loads /en/roomsandrates/1015538 which has a different property (1015538 / hotel 1091113) with a different siteId. Tests the case where child Kitt should destroy top-frame Kitt if they have different sites.

Booking Iframe — No THN Script

Top frame detects Booking Iframe page and starts awakeChildScript(), but the iframe at /en/roomsandrates/9999999 does NOT have the THN script installed. No thn_ping will ever arrive. After 10 seconds the deferral timeout expires: isDeferred becomes false and pageCheck({ force: true }) re-triggers widget evaluation.
